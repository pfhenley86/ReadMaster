# 📚 Read Master – Personal Library Tracker

Read Master is a **C# ASP.NET MVC web application** that allows users to search for books using the Google Books API and manage a personalized digital library.

---

## 🚀 Features

### 🔍 Search Books (Google Books API)

* Search by title, author, or keyword
* View book details before adding

### ➕ Add Books to Your Library

* Save books from search results
* Store key metadata (title, author, description, etc.)

### 📖 Track Ownership & Reading Status

* Mark books as:

* Owned / Not Owned
* Assign reading status:
  * Read
  * Want to Read

### 📚 Library Management

* View all books in your collection
* View individual book details
* Edit book information
* Delete books from your library

---

## 🏗️ Tech Stack

* **Backend:** C# / ASP.NET Core MVC
* **Frontend:** Razor Views (HTML, CSS, Bootstrap)
* **Database:** MySQL
* **Data Access:** Dapper
* **API:** Google Books API

---

## 🧠 How It Works

1. Search for a book using the Google Books API
2. Browse results in the UI
3. Select and add a book to your library
4. The app saves the book to the database
5. Manage your collection:

   * Mark as owned
   * Set reading status
   * Edit or delete anytime

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/your-username/bookmaster.git
```

### 2. Configure app settings

* Add your Google Books API key
* Update your database connection string in `appsettings.json`

### 3. Run the application

```
dotnet run
```

### 4. Open in browser

```
https://localhost:xxxx
```
